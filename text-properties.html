A. Typography Properties: 
      
      1. font-size (normal - 16px)
      2. font-weight (normal - 400, bold - 700)
      3. font-style (italic - i)
      4. Text-transform (uppercase, lowercase, capitalize)
      5. font-family (default - Times Romans - Serif)
      

      B. Spacing and Alignment:
      
      1. Line-height (commonly used)
      2. Letter-spacing
      3. Word-spacing
      4. Text-align
      5. Vertical-Align (super - span, sub, bottom, top)
      6. Direction (rtl - for arabic)
      

      C. Decoration and Effects:
      
      1. text-decoration (underline, line-through)
      2. text-shadow
      3. White-space (no-wrap - for ... by hidding the overflowed text, and using text-overflow property, pre-wrap - acts as pre)

      
      - Extra Tags that could be used with Text-Properties: 
      
      1. width (for shoring the box)
      2. height
      3. margin-bottom or top (usually)
      4. color, background-color
      5. Margins
      6. Padding
