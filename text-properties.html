The following properties can be used for formatting text: 
      1. font-size (normal - 16px)
      2. font-family (default - times romans - serif)
      3. font-weight (normal - 400, bold - 700)
      4. font-style (italic - i)
      5. text-align
      6. line-height vs height 
      7. width (for shoring the box)
      8. margin-bottom or top (usually)
      9. text-decoration (underline, line-through)
      10. color, background-color
